# lock-in-vro
a randomised study prompt generator

## usage
- enter what you are studying for (ex. "maths exam", "science quiz", "english project")
- <img width="492" height="188" alt="image" src="https://github.com/user-attachments/assets/e5a3b2a9-a2db-4e53-bb78-d549e2869b2d" />

- click "Generate"
- The website will give you a motivational prompt and an asthetic image
- <img width="1917" height="966" alt="image" src="https://github.com/user-attachments/assets/38c9abd1-f732-4821-b7b3-d9dedf5ecc16" />

- you my refresh the page to get a different prompt and image each time

## how it works
A simple .random() choice chooses a phrase from a list, and another chooses a template.
User input is used as another factor in the template.
An image is chosen from the open source library 'picsum'
The image is shown with the text overlayed and wrapped.
